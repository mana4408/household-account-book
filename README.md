# 💰 Smart Household Account Book

AI 기반 영수증 자동 입력 기능을 제공하는 스마트 가계부 서비스

## 🚀 주요 기능 (개발 예정)
- 수입/지출 관리
- 카테고리별 통계
- 📸 영수증 OCR 자동 입력

## 🛠 기술 스택
- Java 21, Spring Boot 3.3
- Spring Data JPA
- MySQL, Redis
- Docker, GitHub Actions, AWS
