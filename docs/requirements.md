# 요구사항 명세서

## 1. 프로젝트 개요

- **서비스명**: Smart Household Account Book (스마트 가계부)
- **목적**: 영수증 OCR 자동 입력 기능을 갖춘 개인 가계부 서비스
- **대상**: 개인 사용자

---

## 2. 기능 요구사항

### 2.1 회원 관리
| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 회원가입 | 이메일 + 비밀번호로 가입 | 필수 |
| 로그인 | JWT 토큰 발급 | 필수 |
| 로그아웃 | 토큰 무효화 | 필수 |
| 회원정보 수정 | 닉네임, 비밀번호 변경 | 선택 |
| 회원탈퇴 | 계정 및 데이터 삭제 | 선택 |

### 2.2 카테고리 관리
| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 기본 카테고리 제공 | 식비, 교통, 쇼핑 등 기본값 | 필수 |
| 카테고리 추가 | 사용자 정의 카테고리 생성 | 필수 |
| 카테고리 수정/삭제 | 사용자 카테고리만 수정 가능 | 필수 |

### 2.3 수입/지출 내역 관리
| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 내역 등록 | 금액, 카테고리, 날짜, 메모 입력 | 필수 |
| 내역 조회 | 월별/카테고리별 필터링 | 필수 |
| 내역 수정/삭제 | 본인 내역만 가능 | 필수 |
| 페이징 처리 | 대량 데이터 처리 | 필수 |

### 2.4 통계
| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 월별 수입/지출 합계 | 이번 달 요약 | 필수 |
| 카테고리별 지출 비율 | 파이 차트용 데이터 | 필수 |
| 기간별 추이 | 최근 6개월 트렌드 | 선택 |

### 2.5 OCR 영수증 자동 입력 ⭐ (핵심 차별화)
| 기능 | 설명 | 우선순위 |
|------|------|---------|
| 영수증 이미지 업로드 | JPG, PNG 지원 | 필수 |
| OCR 텍스트 추출 | 네이버 CLOVA OCR API 연동 | 필수 |
| 자동 내역 생성 | 추출된 데이터로 내역 초안 생성 | 필수 |
| 수동 보정 | OCR 결과 사용자가 수정 가능 | 필수 |

---

## 3. 비기능 요구사항

- **보안**: JWT 기반 인증, 비밀번호 BCrypt 암호화
- **성능**: 주요 API 응답시간 500ms 이내
- **확장성**: 도메인별 패키지 구조로 유지보수 용이
- **문서화**: Swagger UI로 API 명세 자동화

---

## 4. 제외 범위 (Out of Scope)

- 소셜 로그인 (Google, Kakao 등)
- 다중 사용자 공유 가계부
- 모바일 앱 (React Native는 추후 고려)
